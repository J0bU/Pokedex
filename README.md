# How to run in dev

1. Clone the repo
2. Run the app
``` yarn install ```
3. Have nestcli installed ``` npm i -g @nestjs/cli```
4. Run the database ```docker-compose up -d```

### Stack used in this project
- MongoDB
- NestJS
